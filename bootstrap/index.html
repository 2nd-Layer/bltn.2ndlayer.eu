Welcome to our Jormungandr HTTP Genesis Block bootstrap service page. With Jormungandr 0.8.11, functionality to fetch block0 out of band using http(s) has been added and we decided to share ours with the world!

To enable this functionality which will not only reduce load on the IOHK trusted peers, but also reduces your bootstrap times, all you need to do is add following code into your node_config.yaml or node_config.json.

JSON configuration

{
  "log": [
    {
      "format": "plain",
      "level": "info",
      "output": "stderr"
    }
  ],
  "http_fetch_block0_service": [
    "https://bltn.2ndlayer.eu/bootstrap"
  ],
  "p2p": {
    "topics_of_interest": {
      "blocks": "normal",
      "messages": "low"
    },
...

YAML configuration

---
log:
- format: plain
  level: info
  output: stderr
http_fetch_block0_service:
- https://bltn.2ndlayer.eu/bootstrap/
p2p:
  topics_of_interest:
    blocks: normal
    messages: low
  trusted_peers:
...
